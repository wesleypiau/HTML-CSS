texto marcado

podemos criar tambem um texto marcado usando a tag "MARK". Para mudar a cor da marcaçao, basta digitar dentro da tag mark dando espaço e colocando a tag style e background color, isso ja se encaixa em css pois cor nao tem significado, cor é forma. ex minha frase agora estara marcada em vermelho

texto grande e pequeno

estamos criando um texto grande e um texto pequeno usando as tags "big" para grande, lembrando que a tag 'big' ja morreu no Html 5, e a tag "small" para pequeno

texto deletado

podemos marcar um texto deletado como excluido, com a tag "del", pra indicar que ele deve ser lido, mas nao considerado

texto inserido

podemos marcar um texto como inserido para o texto ficar sublinhado, para dar uma Ênfase e indicar que ele foi adicionado depois

texto sobrescrito

Para inserir coisas do tipo "x20+3" eu digito a tag sup para "superior", elevar o texto ou numero que quero que fique para cima da linha "normal"

texto subscrito

para inserir coisas do tipo "H2O" posso utilizar a tag "sub" no caracterer que desejo que desça da linha "normal".
