Criei meu conteúdo em HTML, como personalizo meu site?

como fazer o fundo gradiente?

  1. crio um link css
    • lembrando que é preciso criar um link css para toda nova página, para abrir uma página style.css eu clico em cntrl+clique para abrir um novo arquivo.
  2. em #style.css
    • eu crio uma tag body e dentro de { } eu digito a tag background-image:linear gradient, digo em qual posiçao quero o gradient,right, botton, ou deg para fazer com numeros, ex: 45deg e tal, por fim escolho as cores do meu gradient

background-color, width, padding, box-shadow, margin, border-radius

como crio uma caixa de texto de cor diferente do fundo?

  1. digite a tag main
    • a tag main precisa esta dentro de body, já no meu site principal, onde apenas assim eu consigo separar o fundo da minha caixa de texto do fundo da cor do site
  2. background-color
    • dentro de {} de main, eu digito a tag background-color, eu crio a cor da minha caixa de texto
  3. width
    • para selecionar o tamanho da minha caixa, agora ja com a cor que eu escolhi em background-color
  4. padding
    • serve para eu dizer a largura da minha caixa para meu texto "coisa de espaçamento pra ficar harmonico"
  5. box-shadow
    • significa sombra da caixa, nela eu preciso digitar a quantidade de px da sombra da minha caixa de texto, uso 3px´s e a cor desejada da sombra. obs; uso apenas espaço pra delimitar os px´s e espaço tambem para selecionar a cor(nao precisa digitar a tag de cor, apenas digito qual cor quero a sombra) lembrar do ; para fechar.
  6. margin
    • margin é onde eu quero deixar localizada minha caixa, seja no centro, de lado, etc, para o centro uso a tag "margin:auto;"
  7. border-radius
    • serve para eu personalizar a borda da caixa de texto, deixando redondinha, padrão quadrada ou curvada, delimito isso usando porcentagem 10%,20% e ai eu vou vendo na prática atualizando o site e vendo o que fica mais agradavel para os olhos

personalizando fonte de texto e organizaçao do texto

  1. tag P dentro da página #style.css
    • após digitar a tag P, dentro de {} eu digito a tag text-align:justify aqui eu organizo todos os PARAGRÁFOS. atençao mudo apenas a organizaçao dos "PARAGRÁFOS".
  2. fonte geral do site
    • caso eu queira todo meu site com uma fonte só padrão, eu digito essa tag *{font-family: e aqui dentro, a fonte que eu quero que todo meu site tenha}
  3. mudando a cor dos h1,h2 etc.
    • digito a tag h1 e dentro de {} eu digito a tag "color" e escolho a cor de cada h1, h2 etc.
  4. text-shadow
    • serve para sombrear o texto, muito usada para sombrear titulos. dentro das tags, h1, h2 eu uso a tag text-shadow e assim deixo meu titulo sombreado, usando a mesma técnica de sombra usada na box-shadow usando os px´s para delimitar o tamanho e espaçamento das sombras.
  5. centralizando meu titulo
    • basta digitar a tag ainda dentro dos {} de h1 ou h2, text-align:center para deixar meu titulo no centro.
