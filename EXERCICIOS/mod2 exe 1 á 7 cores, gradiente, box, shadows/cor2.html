criando um gradiente na página

para criar uma configuraçao global, para eu poder colocar o degradê de cima para baixo e vice e versa da forma gradiente sem linha quebrando, eu preciso acionar a tag:
*{/*configuraçao global das CSS*/height:100%;} para fixar meu fundo gradiente eu uso a tag:background-attachment:fixed (cuidado, o texto do site pode ficar muito para baixo com essa tag, deve ter atualizado)
caso eu queira mudar a direção basta usar dentro das cores eu uso o angulo com a tag "deg" ex: 45deg para um degradê subindo pra ponta
