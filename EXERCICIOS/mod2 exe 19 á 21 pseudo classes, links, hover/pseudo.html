DIV é um elemento onde eu posso criar caixas para dividir dentro da minha página Html, textos, e destacar com cor quando passo o mouse por cima a caixinha, por exemplo.

como crio uma DIV?

Para criar uma DIV basta abrir dentro da minha página HTML a TAG div e dentro da Tag digitar meu texto.

na mesma página abro uma tag style interna para dar estilo a esta div.

como faço para dar o efeito de cor ao arrastar o mouse?

Basta eu abrir o elemento div:hover e alterar dentro dos {} o meu background color

como faço para alterar individualmente a cor de cada caixa ou texto div:hover?

para quando quiser que a DIV mude de cor quando passar o mouse por cima, basta dentro do texto HTML eu digitar ID dentro da tag DIV e o nome da ID é o que eu quiser, após isso em style, eu usoa tag div#nome da id, feito isso a caixa que eu selecionei mudara a cor pois dentro da tag eu selecionei que meu background color é "tal cor".

Para que serve a o elemento inline-block?

o elemento inline-block é pra separar cada DIV dentro de sua caixa, para dar o efeito de caixa separada, eu digito dentro da tag div em style o seguinte comando: display:inline-block

atençao quando for usar as tags de medidas de altura e largura da caixa como width e height ou border usar a medida "px".
