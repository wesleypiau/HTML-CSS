O que é uma HOVER?

hover é uma pseudo-class que serve como um seletor de objeto, para quando eu passar o mouse emcima gerar uma ação.

como faço para deixar uma foto oculta?

para ocultar qualquer arquivo ou texto, basta eu usar a div como pai daquele arquivo, ex: "div > p" isso dentro de style, fazendo isso: "div > p { display: none} eu escondo o paragráfo que esta dentro da div" caso eu queira usar uma imagem, basta alterar, invés de P uso a tag img, e oculto a imagem, em style fica div>img{ display:none}
