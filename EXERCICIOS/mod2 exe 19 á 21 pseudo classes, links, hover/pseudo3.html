Para que serve a tag, text-decoration?

A tag eu coloco ela dentro de uma hover ou visited, serve para decorar o texto que criei, personalizando, podendo adicionar uma linha "anderline" para toda vez que eu passar o mouse por cima do texto ela ter um efeito de underline.
Ou deixando em "none" para a linha que ja fica fixa padrão quando crio um link de algum site na minha página, para ela desaparecer. em Ingles se escreve (underline)

Para que serve o elemento VISTED?

Serve para eu personalizar um link que ja foi visitado, podendo alterar a cor desse link, adiconar fundo e etc.
