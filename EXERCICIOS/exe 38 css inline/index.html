iniciando a jornada de CSS3

CSS é apenas uma aparencia para página, podendo mudar fontes com a tag "font-family", alterar o local na tela dos meus arquivos, textos, etc. Pondendo inclusive organizar o texto por exemplo, com a tag "style=" "text-align:justify" essa tag especifica organiza o texto para ficar mais aparente ao leitor, para organizar o texto e deixar mais bonitinho sempre quebrando o texto para ficar imprensado.

para o centro, lado e etc basta escolher dentro da tag font=align a localizaçao do texto mais legal para meu site, tudo isso dentro da tag <p> lembrando de colocar sempre dentro das aspas depois do ; a tag.

colorindo minha página

para colorir o fundo da página, basta eu digitar dentro de "body" a tag style= e background-color e dai escolher a cor de fundo, para colorir apenas o meu texto eu clico dentro do meu H1, H2 etc e altero a cor da mesma forma, com style e background color
