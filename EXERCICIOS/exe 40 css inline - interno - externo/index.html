iniciando a jornada de CSS3

css Inline

CSS é apenas uma aparencia para página, podendo mudar fontes com a tag "font-family", alterar o local na tela dos meus arquivos, textos, etc. Pondendo inclusive organizar o texto por exemplo, com a tag "style=" "text-align:justify" essa tag especifica organiza o texto para ficar mais aparente ao leitor, para organizar o texto e deixar mais bonitinho sempre quebrando o texto para ficar imprensado.

para o centro, lado e etc basta escolher dentro da tag font=align a localizaçao do texto mais legal para meu site, tudo isso dentro da tag <p> lembrando de colocar sempre dentro das aspas depois do ; a tag.

colorindo minha página

para colorir o fundo da página, basta eu digitar dentro de "body" a tag style= e background-color e dai escolher a cor de fundo, para colorir apenas o meu texto eu clico dentro do meu H1, H2 etc e altero a cor da mesma forma, com style e background color para colorir o fundo da frase, para colorir a frase é apenas a tag color

css locais / internos

personalize com mais fácilidade

para colorir a página inteira, cada H1, h2 etc, paragráfo ou fonte de texto, basta abaixo de title eu colocar as tags que eu quero alterar, dar "cara" do meu site, para isso basta eu digitar sem pre a tag style, tudo que eu digitar dentro da tag vai ser considerado apenas aparencia do site ex: <style body para mudar todo o corpo do site, após body aparecer na cor amarela da tag eu abro colchetes '{ e digito o que quero personalizar aqui dentro, no caso agora eu quero alterar a cor}'nesse caso eu digitaria background-color que iria alterar a cor de todo o fundo da minha página. No final do texto de código a tag style/> irá esta assim fechada, ai basta colocar todas as personalizaçoes dentro da tag de estilo.

Para alterar a cor do h1, h2 etc, funciona da mesma forma, só ir digitando tudo abaixo (sempre antes do texto do meu do site) nesse caso sempre abaixo de title, alterando a aparencia do meu site nessa forma, eu posso alterar a cor do meu h1 digtando apenas uma vez, e isso irá alterar a cor de todos os meus H1 do meu texto, sem que eu precise alterar um por um. ex: h1{color;} ai vai alterar todos os h1 pra cor que eu escolher, seguindo essa lógica eu posso alterar desde as fontes dos h1, cor, fundo do h1 etc.

e assim todos os meus códigos de personalizaçao da página, ficam tudo acima organizados antes do texto do meu site, sem que eu precise estar alterando cada tag uma por uma.

css externos

aprendendo a criar uma pasta style.css

para que eu possa alterar a aparencia de todas as páginas que eu tiver disponivel, basta apenas eu criar um arquivo "sytle css" em cada página de html nova que eu criar, digitando abaixo de TITLE em cada nova página HTML, a tag link:css, feito isso, eu digito cntrl+clique no nome em laranja "style.css" que vai pedir para criar um novo arquivo/pasta no meu vscode, dentro dessa nova pasta de style css eu posso colocar lá todas as minhas personalizações, alterando lá eu altero a cor, fonte, fundo de todas as páginas, com apenas um clique em cada tag de css.

Dessa forma caso eu queira alterar a aparencia de todas as páginas do meu site, para ficar com todas as páginas iguais, basta apenas eu mexer na página style css que eu criei e modificar a tag que eu quero alterar a aparencia.
