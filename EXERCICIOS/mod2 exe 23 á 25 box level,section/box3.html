qual funçao de cada tag?

em aside aprendemos que sempre é usado para fazer referencia ao meu texto principal.

dentro de article, é onde eu coloco meu titulo, ele é uma tag independente, onde posso criar um novo assunto dentro dessa tag

para que serve as tags, header, section,article,aside, nav, main, footer?

as tags substituem as div´s, facilitando para o navegador a compressão dos códigos, tendo semantica em todas as tags, dando um signficado de pq estou criando essas caixas de texto, no exemplo de header, é onde eu crio meu cabeçalho do site, dentro do meu cabeçalho posso criar um titulo, um paragráfo e todos ele ficaram agrupados em caixa separadas, para quando eu quiser personalizar e deixar cada tag com sua denominada personalizaçao separada uma das outras.

o que é Main?

A tag Main é onde deve ficar o conteudo principal do meu site.

o que é Nav?

Nav, é onde eu posso colocar todos os meus links do site em um local só, podendo em style personalizar os links tudo de uma só vez.

o que é Section

section, é responsavel em tirar o trabalho das div´s, responsavel por marcar as seçoes de conteudo da página, tendo a responsabilidade de poder ter todas as tags dentro dele. ele é responsavel por criar novos assuntos dentro de cada seçao, caso eu queira personalizar depois cada seçao usando uma id para cada seçao eu consigo isso.

para que serve a tag Footer?

serve para dar créditos, rodapé do site ou coisa do tipo.

o que é article?

article, é artigo traduzindo literalmente, serve para voce colocar seu conteudo dentro tendo um signficado mais semantico do que uma section.

o que é aside?

aside, representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais.

quando usar section?

Geralmente uso quando quero colocar em meu site uma foto e dar um contexto a essa foto, ai eu crio um paragrafo para selecionar meu texto ou titulo pra essa foto e em seguida mostro a imagem de que eu estou me referindo, ex:

foto de divulgaçao da WA GESSO 3D

feito isso dentro de style fica mais fácil de personalizar cada tag, com cores, fundo, fonte para cada pedaço de código que criei, é bom para personalizar cada parte separadamente e mudar a cara do site.
