Aprendendo a personalizar

personalizando o titulo

Titulos personalizados, servem para diferenciar um titulo principal do titulo "genérico", podendo mudar a localizado dele na tela, cor, fundo dentre várias personalizaçoes disponiveis, tudo para ficar diferente dos demais titulos

onde usar ID e # ?

ID é usado em HTML para identificar qual titulo eu quero personalizar, pois na pagina de CSS eu posso usar # para dizer ao meu seletor qual propriedade eu quero personalizar, ID em HTML, # em CSS.

qual código usado para mudar o lugar do titulo individualmente?

exemplo abaixo:

Antes de tudo eu crio uma página de CSS, com o link:css. Após isso na minha primeira página eu Digito no meu H1 que quero que seja diferente o seguinte código=> h1 id="principal">. Feito isso na minha página de CSS eu personalizo esse H1 especifico, para ele ficar diferente dos demais H1, usando nessa página de CSS a tag "h1#principal" automáticamente ele vai entender que to falando daquele H1 da página html que eu destaquei, feito isso basta dentro desse "h1#principal" em CSS selecionar a TAG text-align e alinhar esse H1 individualmente em qualquer lugar da tela.

Qual a diferença entre ID e #?

classificando elementos

a diferença de usar ID é que ID é unico, eu não posso usar ele 2x no meu HTML, caso eu queira que todos os meus titulos (individuais) fiquem centralizandos, eu uso CLASS para cada titulo que eu queira personalizar, em CSS ele é identificado por "." e dessa forma eu posso classificar usando Class ao invés de id que ficaria errado repetir o elemento ID

O elemento "Class" serve para personalizar todas as minhas Tag de HTML, podendo repetir sem medo de estar errado, lembrando que quando for referir na página de CSS o elemento Class é conhecido por "." .
